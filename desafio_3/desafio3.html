Desafio javascript clase 3

Se pedirá al usuario que ingrese un número, después mediante un DO se le pedirá que ingrese la operación matemática (suma, resta, multiplicación, división) y con un IF se comprobara si la escribió bien, luego se pedirá el segundo número y se usara un SWITCH para que se ejecute la operación matemática y se devolverá por consola el resultado.
